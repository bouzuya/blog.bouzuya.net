目標がないがしろだった。

競プロ。

AtCoder Problems の Logest Streak を [2022-04-20] の 365 days で放棄した。

ABC247 (+8), ABC248 (-27), ABC249 (+24) の 1198 → 1203 で水・緑・水と変化している。

- <https://atcoder.jp/users/bouzuya/history/share/abc247>
- <https://atcoder.jp/users/bouzuya/history/share/abc248>
- <https://atcoder.jp/users/bouzuya/history/share/abc249>

EPDC を解いていた。しばらく解いていない問題 (最後に解いてから6ヶ月以上あいているもの) を解いている。

プログラミング。朝・夜のコミットの習慣を続けている。 rust365 ([2022-02-19]) は毎日バージョンを上げるのをやめた。 its をつくっている。「完了理由」を追加した。 `its issue finish` の出力を改善するために command / query db 間のやりとりを整備している。 `sqlx の migration` だと Ulid の生成ができないので独自の migration の仕組み `iko` をつくろうとしている。

読書。『ソフトウェアアーキテクチャの基礎』は最初から読み直して 1/3 ほど読んだところ。

運動・体調。朝夜の懸垂・朝のラジオ体操・リングフィットアドベンチャーを継続している。リングフィットアドベンチャーのカスタムモードで称号コンプリートを目指している。新型コロナワクチン接種 3 回目の副反応で寝ていた。気温変化が原因と思う疲れがあった。


ゲーム。 Slay the Spire いくつかの実績を解除した。 Vampire Survivors の 0.5.0 の実績をコンプリートした。

育児。上の子はパズルやマリオなどで遊ぶ。懸垂マシンの最上部までひとりで登る。おおむね同じ時間に寝起きする。下の子は歯が増えて数歩歩けるようになった。

常に `null` を return し useEffect をまとめるためだけの React Component がエージェントによって生成され、「それ React Hooks で良くね？」となった。実態としてそれ自体が何も描画しない Component はあるとしても、せいぜい Context で Component の階層におけるの何かしらのスコープを表しているみたいなものだと思っていたのだけど……。

```typescript
function MyComponent(): null {
  useEffect(() => {
    // ...
  }, []);
  return null;
}
```

[bouzuya/shiori-web] の 2.2.0 をつくった。

shiori-web は以前つくった [bouzuya/shiori-android] というブックマーク管理 をする Android アプリの Web 版。 Rust で書かれた JavaScript-free な Web アプリケーション。

2.2.0 の変更点は次の通り。

- Next / Prev リンクの位置を画面下部に固定
- header にロゴを追加
- footer から header menu に移動
- header menu にバージョン表記を追加

細かい改善を入れた感じ。

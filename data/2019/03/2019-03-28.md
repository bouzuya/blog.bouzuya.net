create-b は [bouzuya/bs][] のための npm initializer 。

0.5.0 および 0.5.1 の変更点は↓。

- 変数 `day` / `utc_day` / `utc_month` / `utc_year` を追加

軽微な変更。変数の追加はキリがないので外部から JSON などで渡す形にするのが良さそう……。

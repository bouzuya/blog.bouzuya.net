書きかたメモ
------------------------------------------------------------------------------

### N min.

minutes: に書くのにかけた時間をつける。

### エントリーの種類

- XXXをつくった       ... プログラミング
- XXX@YYYをつくった   ... プログラミング(バージョンアップ)
- XXXをためした       ... プログラミング(練習・勉強)
- XXXをよんだ         ... 読書
- XXXをみた           ... 映像(映画/アニメ/テレビ etc.)
- XXXをかんがえた     ... アイデア
- 今週のふりかえり(n) ... ふりかえり(nは回)

### 毎週書きたい

- 6 「つくった(プログラミング)」を週1回以上書く
- 8 「つくった(hubot-script)」を週1回以上書く
- 6 「つくった(料理)」を週1回以上書く
- 6 「週のふりかえり」を週1回日曜日に書く

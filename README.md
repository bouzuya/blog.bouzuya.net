書きかたメモ
------------------------------------------------------------------------------

### N min.

エントリーの末尾にはN min.をつける。Nは書くのにかけた時間。

### エントリーの種類

- XXXをつくった     ... プログラミング
- XXX@YYYをつくった ... プログラミング(バージョンアップ)
- XXXをよんだ       ... 読書
- XXXをみた         ... 映像(映画/アニメ/テレビ etc.)

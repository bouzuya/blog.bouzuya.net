# blog.bouzuya.net

bouzuya's blog.

## 書きかたメモ

### N min.

minutes: に書くのにかけた時間をつける。

### エントリーの種類

- XXXをつくった       ... プログラミング
- XXX@YYYをつくった   ... プログラミング(バージョンアップ)
- XXXをためした       ... プログラミング(練習・勉強)
- XXXをよんだ         ... 読書
- XXXをみた           ... 映像(映画/アニメ/テレビ etc.)
- XXXをかんがえた     ... アイデア
- XXXをかった         ... 商品のレビュー
- XXXにでた           ... 勉強会など
- XXXにいった         ... 旅行・勉強会など
- 今週のふりかえり(n) ... ふりかえり(nは回)
